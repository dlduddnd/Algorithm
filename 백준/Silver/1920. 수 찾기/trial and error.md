### 알고리즘
 - 이분탐색문제이다. 이분탐색은
   ```
   while start <= end:
        mid = (start + end)//2

        if l[mid] == tmp:
            print("1")
            break

        elif l[mid] > tmp:
            end = mid - 1
            

        else:
            start = mid + 1
   ```
   이렇게 구현해줄 수 있다

### 오류
 - 처음에 단순하게 in함수를 써봤지만 어림도없이 시간초과가 발생했다.
 - 우선 M개의 수들을 정렬을 해서 출력오류가 발생했었다.
 - 0을 출력하는 코드를 어느 구간에 넣어야할지 헷갈려 디버깅이 오래걸렸다.
